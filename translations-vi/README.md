# VN-Office — Vietnamese UI translation additions

This folder holds Vietnamese (`vi`) translation files from LibreOffice's
`translations` submodule that VN-Office has completed/extended, so the menus,
menu commands and tooltips display 100% in Vietnamese.

- `officecfg/.../Office/UI.po` — command labels, context-menu labels and
  tooltips. VN-Office filled in the previously-untranslated entries (menus,
  commands and their tooltip explanations) so this file is fully translated.

These apply on top of the upstream LibreOffice `translations` repository at the
base commit referenced in the top-level README.
